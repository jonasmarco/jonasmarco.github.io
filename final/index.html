fancyBox3

fancyBox é uma biblioteca JavaScript usada para apresentar imagens, vídeos e qualquer conteúdo html de forma elegante. Tem todos os recursos que você esperaria - toque habilitado, responsivo e totalmente personalizável.

Acesse - http://fancyapps.com/fancybox/3/

Inclua os arquivos jquery.fancybox.min.css e jquery.fancybox.min.js no seu projeto juntamente com a versão mais recente do jQuery.

fancyBox3

FancyBox é uma biblioteca JavaScript usada para apresentar imagens, vídeos e qualquer conteúdo html de forma elegante. Tem todos os recursos que você esperaria - toque habilitado, responsividade e totalmente personalizável.

Acesse - http://fancyapps.com/fancybox/3/

Inclua os arquivos jquery.fancybox.min.css e jquery.fancybox.min.js no seu projeto juntamente com a versão mais recente do jQuery. Usar uma CDN também é uma alternativa
